# Des IA izar el diseño de producto

Presentación web local para una charla de 35 minutos y 5 minutos de preguntas.

## Abrirla

Abre `index.html` en Chrome y pulsa `F` para pantalla completa. Se controla con las flechas o espacio. Las teclas del 1 al 5 llevan a cada sección. La tecla `O` abre el recorrido.

## Apertura en Miro

Los minutos iniciales son el árbol de oportunidades que aparece en las diapositivas 3 y 4. Para recrearlo en Miro usa este contenido:

Centro: Resolver rápido no es diseñar mejor.

Crédito: momento, aspiración y lenguaje de la decisión.

Seguros: claridad, comparación y una elección que se siente propia.

Vivienda: señales de intención antes de pedir esfuerzo.

Inventario: ritmo, entorno y feedback al registrar.

Pregunta para cerrar el tablero: ¿Qué momento humano está ignorando la solución más obvia?

## Recorrido de la charla

La presentación abre con el problema, pasa por el árbol de oportunidades y enseña el filtro Persona, Postura y Precisión. Cada capa tiene ejemplos para los cuatro retos, reglas de ejecución con Tailwind y un prompt breve. El cierre propone una pausa de quince minutos para revisar el demo antes de seguir construyendo. El recorrido y los tiempos de exposición están en `guion.md`.
